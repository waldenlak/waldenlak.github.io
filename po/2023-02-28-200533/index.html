第三方插件

主流推荐的有 Advanced Tables、Dataview

写作增强

  • 💜editing-toolbar：添加一个类似 Word 的浮动工具编辑栏，达到富文本编辑的效果，减轻热键或命令的记忆负担。
  • 增强编辑
  • 💜Table Enhancer 实时预览模式下直接操作 md 表格：右键单击菜单中的 Create new table 生成表格，单击单元格直接编辑，按 ↓(下) ↑(上) tab(右) Shift + Tab(左) 在单元格之间移动，按 Enter 或 Esc 或单击表外的任意位置退出编辑模式。
  • obsidian-excel-to-markdown-table：可以将来自 Microsoft Excel、Google Sheets、Apple Numbers 和 LibreOffice Calc 的数据粘贴为 Obsidian 编辑器中的 Markdown 表格。
  • 💜Outliner：像 Workflowy 或 RoamResearch 的大纲模式一样，使用快捷键调整无序列表项目的顺序和层级。
    • Tab(缩进为子级) Shift + Tab(提高为父级)
    • Cmd/Ctrl + Shift + ↑ 以及 Cmd/Ctrl + Shift + ↓ 移动整个树节点
    • 鼠标拖拽节点图标移动节点。点击左侧图标收起/展开节点。
  • url-into-selection：使用常规 Ctrl/Cmd+V 将链接插入到选定的文本中，或反过来，通过命令面板/快捷键将文本插入到选定的链接中。
  • obsidian-auto-link-title：自动在粘贴时提取网页链接标题，创建一个 Markdown 链接。相当于后台打开网页提取标题，速度慢的要等好几秒。
  • Obsidian 插件：Floating TOC 为你添加悬浮的笔记目录

笔记管理

  • 💜file-tree-alternative：类似 Evernote 的增强型文件管理器面板，同时显示文件夹和文件列表、聚焦于某个文件夹、文件/数量统计
  • 💜💜Linter：YAML 数据预设与格式规范 (自动补全缺少的属性)，Markdown 格式规范
  • 💜💜Obsidian 插件：Projects 提供笔记多视图的管理能力：在左侧栏点击 Open projects 的图标进入到 Projects 的窗口，以表格、看板、日历、画廊等多种视图来校对 YAML 属性数据。支持按条件筛选过滤显示文件、按条件标记不同颜色，设置筛选条件来为不同数据进行颜色。
  • 💜使用 Templater＋QuickAdd 自动套用模板新建文章，自动生成 YAML 属性，组合命令形成自动化流程👉[Obs＃55] 建立新筆記的模板設定-Calendar, Templater與QuickAdd – 簡睿隨筆
    • Templater 语法解释与示例👉tp.system - Templater；Templater Snippets - shabeblogHexo静态博客配置与管理发布流程 | SineObama的博客
    • [Obs＃51] QuickAdd全攻略(2)：脚本撰写与宏使用要点
  • dataview ：索引 md 文件，并创建复杂的查询视图如表格 列表 任务 日历，对于处理大量信息的复杂项目、管理项目清单或进行定量分析非常有用（但对于普通用户来说过于硬核）。
    • 多个层次的深度汇总👉Dataview plugin snippet showcase - Share & showcase - Obsidian Forum
    • 可视化 frontmatter，使用复选框跟踪进度👉How to use Obsidian’s Dataview plugin to visualize frontmatter
