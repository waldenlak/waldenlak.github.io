一、官方网站

  • 1.1、免费帐户：ChatGPT网页 访问需要魔法（香港节点不可用）中国区注册 OpenAI 账号
  • 1.2、Plus 帐户：ChatGPT Plus 开通教程。
  • 1.3、API Key：可用于第三方 ChatGPT 应用程序，功能与 ChatGPT 完全一样，且国内网络能正常使用。优势在于自定义和灵活性。ChatGPT 会赠送注册用户 5 美元的额度，但不管用不用，1 个月后都会被清零收回。①官方 API Key 查询 ②查询 API Key 余量 ③免费 API Key 1 ④免费 API Key 2 ⑤免费 API Key 3
    • API Key 收费：ChatGPT Plus 采用每月 20 美元的收费方式，但对于轻度使用的用户来说，ChatGPT API 的方式更为灵活。每 0.002 美元 1000 Tokens，每 100 万单词 18 元人民币，每 100 万汉字 28.5 元人民币，每 1 元人民币约 3.5 万汉字 。GPT-4 的 Token 消耗，根据使用情况，是 GPT-3 的 15 倍 和 30 倍，而且必须是 Plus 付费用户。
    • Token 说明：指一段话里被分出来的词汇，如 I love you 是 3 个 Tokens。而其它语言需要先转化 Unicode 后再算 Token，如我爱你是 11 个 Tokens。①官方 Token 计算 ②Tiktokenizer ③InvertedStone。
  • 1.4、合租代买：国内网络购买 ChatGPT 服务比较困难，OpenAI 官方发布的 iOS 应用也会频繁遇到购买失败的情况。退而求其次的方法是找第三方合租代买，当然价格会比原价高，这样的店铺有很多，如 银河录像局、奈飞小铺、蜜糖商店，特点是国内有备案，跑路概率更低。

三、GPT-4 镜像

2023 年 3 月 14 日，OpenAI 发布 GPT-4。相比之下，GPT-4 的回答准确性不仅大幅提高，还具备更高水平的识图能力，输入限制也提升至 2.5 万字。目前 GPT-4 只能通过 Plus 帐号体验，价格 19.9 美元/月，而且国内帐户无法直接购买。下面我找了一些 GPT-4 替代方案。

  • 3.0、New Bing：微软自称已在 New Bing 中集成 GPT-4，但实际效果不可知。
  • 3.1、ChatmindAI：国内站长开发的 ChatGPT 网站，特点是无需魔法即可使用，需要注册使用，注册用户可每天免费使用 2 次。
  • 3.2、Forefront Chat：国外最近上线的一款在线机器人聊天网站，其底层应用是基于 Vercel.com 的 AI Playground。它集成了 GPT-3 和 GPT-4（输入框右边图标可切换），还内置了 68 个知名人物，让你的聊天各不相同。实际测试，登陆后可以使用 GPT4（每 3 小时 5 条消息）
  • 3.3、Poe：国外知识问答网站 Quora 基于多个第三方大型语言模型训练的 AI 聊天应用，顶部有多个机器人名称。用户每天免费使用 GPT-4 一次和 Claude+ 三次。值得一说的是，Poe 支持国内双币种信用卡。
  • 3.4、Ora.sh：基于用户指定的 prompt，可以创建用于不同场景的聊天机器人。在 Ora Explore 页面上有很多基于 GPT-3.5 的聊天机器人，并且不必登录便可以访问。同时，Ora.sh 也提供了GPT-4 的试用。
  • 3.5、GPT4free ：一个由欧洲计算机系学生开发的项目，项目聚合了众多第三方 ChatGPT 应用的免费 GPT-4 接口。包括 Forefront.ai、Poe.com、BingChat 等网站。接口是通过“逆向工程”获取的，然后需要在 Discord 上面使用（免费），算得上是目前最热门的 GPT-4 项目。
    • 使用方法：首先，加入 GPT4free 群组，然后，点击左侧的「ai-chatroom」进入聊天，使用指令「!Ai 模型名称 聊天内容」进行聊天。支持的模型包括：ChatGPT、GPT-4、 Sage、Claude、Claude+、 Dragonfly 等。比如输入：!Ai GPT-4 你是GPT4吗？ 输入后，当输入框显示「Freegpt4 正在输入….」则说明正在回复中。而 GPT-4 模型花费的时间会根据你问的内容来决定，简单的问题几秒就能给出答案，如果要 1000 字时，花费的时间可能会超过 1 分多钟。期间静静等待即可，不要重复输入相同的内容，实在等不及了，可以刷新一下网页。
  • 3.6、Poe Token：GPT4free 里还有一个叫 poe-token 的频道，这是 Poe 帐号共享的一个项目。
    • 使用方法：点击左侧的「#poe-token」进入频道，这里会显示实时分享的 Tokens，任意选择复制一行即可。接着，给浏览器安装上扩展 Cookie-editor（地址：Chrome、Edge、Firefox、本地下载）。再来，打开网站 https://poe.com，点击扩展图标，将网页 Cookie 修改为之前复制的 Tokens。最后，点击保存，刷新一下网页即可使用。此外，你可以左侧菜单里的“Settings”，查看当前帐号的可用额度，如果没有了，那就更换一个新的 Tokens。

四、网页应用

原版 ChatGPT 不仅有访问限制，而且没有辅助功能，于是网友开发了各种增强版的 ChatGPT 网页应用。注意，大多数情况下，下面的应用都需要先填写 API Key。

  • 4.1、TypingMind：国外一名网友实在不喜欢 ChatGPT 打字慢和频繁登录开发的增强版 ChatGPT 。本质上，它是一个基于 ChatGPT API 的应用，但它有更好的用户界面和功能，包括聊天搜索、提示词库、网络搜索、上传文档等功能。
  • 4.2、ChatKit：另一款基于 OpenAI API 的增强版 ChatGPT。它拥有 TypingMind 的所有功能，除了可以上传文档来聊天，它还可以输入网页 URL 来做为上下文素材。
  • 4.3、Chatbot UI：开源，基于 ChatGPT API 的网页应用，界面与原版很像，可自定义提示词，支持数据导入导出。
  • 4.4、Chat Chat：另一款开源 ChatGPT API 网页应用。支持的功能十分丰富
  • 4.5、MyGPT：一款基于 ChatGPT 的第三方 ChatGPT 网页应用，独特之处在于支持 ChatGPT 的插件功能。而且，还内置了提示词库，支持聊天记录和搜索等功能。遗憾的是，网站默认使用英文，而它自带的翻译效果不是很理想。
  • 4.6、自建 ChatGPT
    • ChatGPT Web - 演示网站，第三方镜像网站最常用的项目。
    • ChatGpt-Web - 演示网站，使用 React 搭建，还支持绘画。
    • ChatGPT-Next-Web - 演示网站，奶酪搭建的网站：https://cheesechat.vercel.app
    • Chatgpt Demo - 演示网站，速度飞快
    • Chuanhu ChatGPT - 演示网站，可直接在 Hugging Face 上部署。

五、桌面应用

  • 5.1、灵动 AiChat：国内站长开发的免费 ChatGPT ，有 网页版（密码：666）和 客户端 版本，无需登录，国内网络就可以直接使用。本质上，这款软件就是将网页版嵌入到客户端里，所以它可以动态更新网页，也就不会失效了。支持 Windows、macOS、Android 三个平台。
  • 5.2、OneGPT： 一款聚合了 ChatGPT、文心一言、通义千问、POE、Bard、New Bing 等多个平台的客户端。软件也内置了免费的 ChatGPT 镜像网站，同时你可以使用自己的 API Key。支持 Windows、macOS 和 Linux。
  • 5.3、ShellGPT：一款聚合了多个网站的免费 ChatGPT 客户端，无需登录，多节点自动切换，还支持免翻 Bing。内置了聊天、翻译、AI 作图等多种模式，支持 Windows、macOS、Linux、Android。
  • 5.4、ChatGPT Desktop：ChatGPT 发布以前最早开发的桌面客户端之一。使用 Rust 语言编写，本质上就是将 ChatGPT 网页版打包为桌面版，支持 Windows、Linux、MacOS 多平台。使用时需要使用魔法。
  • 5.5、Chatbox：一款开源跨平台的 ChatGPT 客户端。需要填入 API Key 才能使用，特点是内置了提示词和与管理工具，支持 Windows、Mac 和 Linux 多平台。
  • 5.6、ChatALL：一款聚合了多家语言模型的桌面客户端，可以把一条指令同时发给多个 AI，帮助您发现最好的回答。支持 Windows、macOS 和 Linux。
  • 5.7、MacGPT：一款免费的 macOS 桌面 ChatGPT 客户端。支持网页 和 API 两种方式，支持全局调用，也支持在菜单栏上调用，还可以语音输入。采用自愿付费的方法，你想免费下载，只需要输入 0 元，即可下载，但只支持 macOS 12+。 iOS 客户端叫 GeePeeTee

六、手机应用

  • 6.1、iOS OpenAI Chatgpt，目前只能通过 App Store 外区帐号下载。
  • 6.2、NewBing：微软推出的基于 ChatGPT 的搜索应用，可以在 Edge 浏览器使用，支持 iOS 和安卓平台。iOS Bing Search 同样也需要外区 ID 下载。此外，你还可以通过修改 UA，来让任意浏览器都可以使用 New Bing。
  • 6.3、iOS 版 Poe
  • 6.4、GeePeeTee 与桌面客户端 MacGPT 出自同一家，支持 iOS，只需输入 ChatGPT 的 API Key 即可开启聊天。
  • 6.5、问东风 AI：国内用户开发的 iOS 套壳 ChatGPT，无需设置就可以使用，而且免费无广告。类似应用：ChatAI Lite
  • 6.6、灵动 AiChat：国内站长开发的免费 ChatGPT ，有 网页版（密码：666）和 客户端 版本，无需登录，国内网络就可以直接使用。本质上，这款软件就是将网页版嵌入到客户端里，所以它可以动态更新网页，也就不会失效了。支持 Windows、macOS、Android 三个平台。
  • 6.7、ChatGPT 在线：由国内开发者基于 ChatGPT 开发的安卓应用，国内可免魔法直接使用，有问答、聊天、写作、绘画 4 大功能。免费用户每天免费使用 20 次。类似应用：AIChat 中文，使用时需要登录用，但注册时可以随便填一个手机号就行。

七、大语言模型

  • Sage：更擅长英语以外的语言。
  • Claude：专注于英语，更擅长于创意写作任务。
  • Claude+：优于 Claude，尤其是在非英语语言方面。
  • Claude 100K：支持一性次输入 75000 个单词，远超 GPT-4 32K 的 25000 个单词。
  • Claude-instant：Claude 的超快响应版。
  • Dragonfly：响应内容较短，因为它使用的是 text-davinci-003 模型。
  • ChatGPT：原版 ChatGPT
  • GPT-4：目前可用的最强大语言模型。

随着 ChatGPT 的大火，众多公司也入局了 ChatGPT，推出了自己的大语言模型，这其中不乏有很多佼佼者。

  • 7.1、Claude：原 OpenAI 员工离职后自立门户开发的 AI 机器人，效果媲美 ChatGPT，被称为 ChatGPT 的最强对手。注册一个 Slack 帐号，如果有谷歌帐号可直接授权登录，然后创建一个“工作区”；搜索并添加 Claude 到 Slack，即可使用。
  • 7.2、Bard：由谷歌推出的类 ChatGPT 应用，目前已经开放使用，但还是需要申请才能使用。与 ChatGPT 相比，Bard 还不支持代码功能，也不支持中文。
  • 7.3、Vicuna：由斯坦福学者联手 CMU、UC 伯克利等推出的一个全新模型，俗称「小羊驼」。经测试，Vicuna 已经能达到 ChatGPT 90% 的性能。
  • 7.4、文心一言：由百度推出的大语言模型，也是继 ChatGPT 之后最早入局的，现已全面开放，但仍然需要申请才能使用。除此之外，文心一言还支持绘画、视频生成等功能
  • 7.5、通义千问：由阿里推出的大语言模型，现已开放申请，申请时需要填写企业名称，但实测随便写一个公司名称也可以通过。上手体验后感觉很不错，有望成为国内最好的大语言模型。
