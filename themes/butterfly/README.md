```powershell

```

## Config

Set theme in the hexo work folder's root config file`_config.yml`:

```yml
theme: bud
```

- If you don't have `pug` and `stylus` rendererï¼Œplease install: [hexo-renderer-pug](https://github.com/hexojs/hexo-renderer-pug) or [hexo-render-pug](https://github.com/maxknee/hexo-render-pug), and [hexo-renderer-stylus](https://github.com/hexojs/hexo-renderer-stylus):

In Hexo site root directory
