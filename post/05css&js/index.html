一、css/js自制样式

方法：
在theme/butterfly/config.yml内的inject项插入css或js格式的DIY网站样式代码，便于快速升级主题。
参考教程：Custom Beautify | Akilarの糖果屋

css

  • 字体样式修改
  • 版块显隐修改
  • 修改卡片透明度、配色、圆角等元素样式
  • 日间模式、夜间模式或阅读模式修改主题色与配色等全局页面样式
  • 鼠标指针样式替换
    自定义图标——css引入阿里iconfont图标（2022年6月开始iconfont无定期维护，可选择其他图标库）：Iconfont Inject | Akilarの糖果屋

js

  • 设置首页随机背景或banner效果
源码

\source\js\randombg.js

  • 设置浏览器标签页处站点动态标题
